Navigation / Controls

Modes
V = Select
S = Scale
M = Measure

Navigation
Ctrl + Mouse Wheel = Zoom to Cursor
Shift + Middle Mouse Drag = Pan
Mouse Wheel / Scrollbars = Scroll

Editing
Click line = Select, only while in Select mode
Shift while drawing = Lock to closest X/Y axis
Angular snap toggle = Snap to chosen angle step
Measurement grid toggle = Show grid using real units
Snap measures to grid = Snap measurement endpoints to grid
Delete / Backspace = Delete Selected
Ctrl+Delete = Clear Measurements
Ctrl+Backspace = Clear All / Reset
Ctrl+Z = Undo
Ctrl+Y / Ctrl+Shift+Z = Redo

Menus
Drag right border of menu = Resize sidebar
Drag bottom-right of each menu island = Resize island
Side arrow = Hide / show menu

Image Input
Paste Image button
Ctrl+V Paste
Drag & Drop Image

Measurements

Yellow = scale. Blue = measurement. Drag point to point.
